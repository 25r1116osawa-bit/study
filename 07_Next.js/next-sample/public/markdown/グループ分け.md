## ■ ページ上部のヘッダー部分：headerタグ

ページ上部に配置される要素（ロゴ、ページタイトル、ナビゲーションなど）をまとめるタグ。
※ HTML の head とは全く別物なので注意。

例：
```
<header>
    <h1>ページタイトル</h1>
    <p>このWebサイトはWeb業界の最新情報をお届けしています。</p>
</header>
```
## ■ ナビゲーションメニュー：nav タグ

主なページ内リンク（メインメニュー）をまとめるタグ。
多くの場合 header の中に置く。

例：
```
<header>
    <h1>ページタイトル</h1>
    <nav>
        <ul>
            <li><a href="#">サービス紹介</a></li>
            <li><a href="#">料金</a></li>
            <li><a href="#">お問い合わせ</a></li>
        </ul>
    </nav>
</header>
```
## ■ 独立した記事部分：article タグ

ブログ記事・ニュース記事など、それ単体で内容が完結する部分に使う。

例：

```
<article>
    <h2>記事タイトル</h2>
    <p>最新のスマートフォン情報！新機種が発表されましたね！</p>
</article>
```

## ■ テーマ別の区分：section タグ

特定のテーマに関連したまとまりを作るタグ。
単体では完結しない内容に使う（article ほど独立していない）。

例：

```
<section>
    <h2>その他のおすすめ記事</h2>
    <ul>
        <li><a href="#">スマートウォッチって使える？使えない？</a></li>
    </ul>
</section>
```

## ■ ページの主要内容：main タグ

そのページの 一番重要なコンテンツ全体を囲むタグ。
中には article や section が複数入る場合がある。

例：
```
<main>
    <article>
        <h2>記事タイトル</h2>
        <p>最新のスマートフォン情報！新機種が発表されましたね。</p>
    </article>

    <section>
        <h2>その他のおすすめ</h2>
        <ul>
            <li><a href="#">スマートウォッチって使える？使えない？</a></li>
            <li><a href="#">ランニングにはこのガジェットがおすすめ</a></li>
        </ul>
    </section>
</main>
```

## ■ 補足情報：aside タグ

メインコンテンツとは直接の関連性が薄い、補助的な情報に使う。
広告、プロフィール、関連リンクなどに向いている。

例：
```
<aside>
    <p>この記事を書いた人：Webガジェット研究家 山田</p>
</aside>
```
