## HTMLフォーム
- ウェブサイトでユーザーから情報を集め、それをサーバー（サイトの裏側）に送るための機能です。
- 例：会員登録、ログイン、お問い合わせ、注文。
<br><br>
### フォームの基本的な構成要素
form タグ：フォームを作成するためのタグで、フォームで使用するすべてのパーツを&lt;フォーム&gt;で囲みます。

|属性|用途|
|---|---|
|action|データの送信先ページを指定|
|method|データの転送方法の指定、主にgetかpostを入力|
|name|フォームの名前を指定|

例）
```
<form action="データの送信先ページのURL" method="get" name="test">
```

ポイント：以降で紹介するパーツはHTMLやCSSで動作するものではなくPHPなどでデータの受け渡しを行うが、HTMLCSSの範囲外であるため、割愛

<br><br>
### フォームで使うパーツ

1. input：テキストやメール、電話番号などの入力欄
2. textarea：複数行のテキスト入力
3. select / option ：選択式入力

共通ポイント：name 属性が必須（サーバーや送信先でデータを識別するためのキー）

<br>

#### 1.input OOO

&lt;input type="○○"&gt;
| `type`     | 表示される部品  | 説明                     |
| ---------- | -------- | ---------------------- |
| `text`     | テキスト入力欄  | 名前・住所など一般的な文字入力。       |
| `password` | パスワード欄   | 入力が `●` や `*` で隠される。   |
| `number`   | 数値入力     | 数字だけ入力できる。矢印で増減可能。     |
| `tel`      | 電話番号欄    | 電話番号に最適化（スマホで数字キーボード）。 |
| `email`    | メールアドレス欄 | メール形式か簡易チェックが入る。       |
| `url`      | URL入力欄   | URL形式チェックが入る。          |

🟢 日付・時間入力
| `type`           | 表示される部品      | 説明                  |
| ---------------- | ------------ | ------------------- |
| `date`           | カレンダー形式の日付入力 | 年・月・日を選択。           |
| `month`          | 月選択          | 年月だけを選ぶ。            |
| `week`           | 週選択          | 年＋週番号を入力。           |
| `time`           | 時刻入力         | 時刻を選択。              |
| `datetime-local` | 日付＋時刻入力      | 日付と時間を同時入力（ローカル形式）。 |


🟠 選択・オンオフ系
| `type`     | 表示される部品  | 説明            |
| ---------- | -------- | ------------- |
| `checkbox` | チェックボックス | 複数選択可。        |
| `radio`    | ラジオボタン   | 一つだけ選択。       |
| `range`    | スライダー    | 数値をつまみで調整できる。 |


🟣 ファイル・画像など
| `type`  | 表示される部品   | 説明                 |
| ------- | --------- | ------------------ |
| `file`  | ファイル選択ボタン | 画像・PDFなどを選べる。      |
| `image` | 画像ボタン     | 押すとフォーム送信される画像ボタン。 |

🟡 ボタン系
| `type`   | 表示される部品 | 説明                   |
| -------- | ------- | -------------------- |
| `submit` | 送信ボタン   | フォームをサーバーに送信。        |
| `reset`  | リセットボタン | 入力内容を初期状態に戻す。        |
| `button` | 汎用ボタン   | JavaScriptと組み合わせて使用。 |

### HTMLフォームから外部サービスに直接送信する設定

カスタムフォーム（Custom Form）を外部API／サービスに接続する。
| 特徴                                    | 説明                                        |
| ------------------------------------- | ----------------------------------------- |
| `name="entry.数字"`                     | Google フォームの各質問に割り振られた **内部エントリID** に合わせる |
| `<form action=".../formResponse">`    | Google フォームの **送信エンドポイント** を指定            |
| `<input>` / `<select>` / `<textarea>` | 選択肢や入力値を Google フォームに送信                   |
| method="POST"                         | データをフォームに送信（GETでもできるがPOSTが一般的）            |


HTML Code例
```
<body>
    <form id="contact-form" class="theme-light" action="googleフォームのアドレスを入力">
        <h2>お問い合わせフォーム</h2>
        
        <label for="category">🍶 お問い合わせ種別 <span class="required">(必須)</span></label>
        <select id="category" name="entry.1352168051" required>
            <option value="">選択してください</option>
            <option value="ご予約・ご予約変更・キャンセル">ご予約・ご予約変更・キャンセル</option>
            <option value="店舗・サービスに関するご意見">店舗・サービスに関するご意見</option>
            <option value="宴会・貸切に関するご相談">宴会・貸切に関するご相談</option>
            <option value="採用に関するお問い合わせ">採用に関するお問い合わせ</option>
            <option value="その他">その他</option>
        </select>

        <label for="name">👤 お名前 <span class="required">(必須)</span></label>
        <input type="text" id="name" name="entry.1453215926" required placeholder="山田 太郎">

        <label for="email">📧 メールアドレス <span class="required">(必須)</span></label>
        <input type="email" id="email" name="entry.598243994" required placeholder="example@mail.com">

        <label for="tel">📞 電話番号 <span class="required">(必須)</span></label>
        <input type="tel" id="tel" name="entry.1001390762" required placeholder="09012345678" pattern="[0-9]{10,11}">

        <label for="message">📝 お問い合わせ内容 <span class="required">(必須)</span></label>
        <textarea id="message" name="entry.1795522726" rows="8" required placeholder="具体的な内容をご記入ください。"></textarea>

        <input type="checkbox" id="policy" name="policy" required>
        <label for="policy">
            <a href="policy.html" target="_blank">プライバシーポリシー</a>に同意します <span class="required">(必須)</span>
        </label>
        
        <button type="submit" id="submit-button">送信する</button>
    </form>
</body>

```
