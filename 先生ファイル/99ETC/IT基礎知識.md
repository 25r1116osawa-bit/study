# ネットワークの仕組み

## 🌐 インターネットとネットワークの基本

### LAN（ラン）とWAN（ワン）

- **LAN (Local Area Network):**
  - 学校や会社、家の中など、**限られた範囲**（Local）のネットワークです。
  - 同じ建物内のプリンタに接続したり、PC間でファイルを共有したりするのに使われます。
- **WAN (Wide Area Network):**
  - 離れた場所にあるLAN同士をつなぐ、**広範囲**（Wide）のネットワークです。
  - 東京本社と大阪支社をつなぐネットワークなどがこれにあたります。
  - **インターネット**は、世界中のネットワークをつないだ、最も巨大なWANの一種です。

### Wi-Fi（ワイファイ）

- LAN（ローカルネットワーク）に**無線（電波）**で接続する技術のことです。
- ケーブルが不要なため、スマートフォンやノートPCなど、様々な機器を気軽にネットワークに接続できます。

## 🖥 インターネットを支えるルール

### TCP/IP（ティーシーピー・アイピー）

- インターネットで通信を行うための**基本的なルール（プロトコル）**のセットです。
- 世界中のコンピュータが同じルールでやり取りすることで、インターネットが成り立っています。
  - **IP (Internet Protocol):** データを目的地（IPアドレス）まで運ぶルール。
  - **TCP (Transmission Control Protocol):** データを正しく、確実に届けるためのルール（届いたか確認したり、順番通りに並べ替えたりする）。

### IPアドレスとDNS（ディーエヌエス）

- **IPアドレス (Internet Protocol Address):**
  - ネットワーク上の機器（PCやスマホなど）に割り当てられる、**インターネット上の住所**のようなものです。（例: `192.168.1.1` や `172.217.26.227`）
  - この住所があることで、目的の機器にデータを届けることができます。
- **DNS (Domain Name System):**
  - `google.com` のような**ドメイン名（人間が分かりやすい名前）**を、`172.217.26.227` のような**IPアドレス（コンピュータが分かる住所）**に**翻訳（名前解決）**する仕組みです。
  - 私たちは普段、IPアドレスを意識せず、ドメイン名でWebサイトにアクセスできています。

# コンピュータの知識（ハードウェア）

## 🧠 PCの主要パーツ

コンピュータは、人間でいう「脳」「作業机」「記憶」などに例えられる部品で構成されています。

### CPU（シーピーユー）

- **Central Processing Unit** の略で、「中央処理装置」とも呼ばれます。
- コンピュータの**「脳」**にあたる部分です。
- あらゆる計算や処理、他の部品への命令を行います。
- CPUの性能が高いほど、コンピュータの処理速度が速くなります。

### メモリ (RAM)

- **Random Access Memory** の略。
- CPUが作業を行うための **「作業机（デスク）」** のようなものです。
- 電源が入っている間だけ、処理中のデータやプログラムを一時的に記憶します。
- メモリ容量（作業机の広さ）が大きいほど、一度に多くの作業（複数のソフトを同時に開くなど）を快適に行えます。
- **電源を切ると内容は消えてしまいます。**(揮発性メモリ)

### ディスク（ストレージ）

- データやプログラムを**長期間保存**しておくための **「記憶装置（本棚や倉庫）」** です。
- 電源を切っても内容は消えません。(不揮発性メモリ)OS、作成したファイル、インストールしたソフトなどはここに保存されます。
- **HDD (Hard Disk Drive):**
  - 磁気ディスクを回転させて読み書きします。
  - 安価で大容量なものが多いですが、衝撃に弱く、読み書き速度はSSDより遅いです。
- **SSD (Solid State Drive):**
  - 半導体メモリ（USBメモリなどに近い）を使って読み書きします。
  - HDDより高価ですが、**読み書きが非常に高速**で、衝撃に強く、静かです。最近のPCの主流です。

### RAM（ラム）とROM（ロム）の違い

- **RAM (Random Access Memory):**
  - 上記「メモリ」のこと。
  - **読み書き可能**ですが、**電源を切ると消える**（揮発性メモリ）。
  - 用途：一時的な作業場所。
- **ROM (Read Only Memory):**
  - **読み取り専用**のメモリです。
  - **電源を切っても消えない**（不揮発性メモリ）。
  - 用途：PCの起動に必要な基本的なプログラム（BIOS/UEFI）など、書き換える必要のない重要な情報の保存に使われます。

# OSの基本操作（Windows）

OS (Operating System) は、コンピュータ全体を管理し、人が使えるようにする基本的なソフトウェアです。（Windows, Mac OSなど）

## 🖥 デスクトップと基本要素

- **デスクトップ:**
  - PC起動後に表示される基本的な画面です。作業机の天板のようなもので、アイコンやショートカットを置けます。
- **タスクバー:**
  - 通常、画面下部にある帯状の領域です。
  - 起動中のアプリが表示されたり、よく使うアプリをピン留め（固定）したりできます。
- **スタートメニュー:**
  - タスクバーの左端にあるWindowsマークをクリックすると表示されます。
  - インストールされているアプリを探したり、PCの設定を開いたり、電源を切ったり（シャットダウン）する時に使います。

## 🖱 ウィンドウの操作

- アプリを起動すると、通常「ウィンドウ」と呼ばれる枠の中に表示されます。
- **最小化（ \_ ）:** ウィンドウを一時的にタスクバーに格納します。
- **最大化（ □ ）:** ウィンドウを画面全体に広げます。
- **閉じる（ × ）:** ウィンドウ（アプリ）を終了します。
- **ドラッグ:** ウィンドウの上部（タイトルバー）をマウスで掴んだまま動かすと、位置を移動できます。

## 📂 ファイルとフォルダの操作

- **ファイル:**
  - テキスト文書、画像、動画、プログラムなど、情報のかたまりです。
- **フォルダ:**
  - ファイルを整理・分類するための「入れ物（箱）」です。フォルダの中に別のフォルダを作ることもできます。
- **主な操作（対象を右クリック）:**
  - **新規作成:** 何もない場所で右クリックし、新しいファイルやフォルダを作成します。
  - **名前の変更:** ファイル名やフォルダ名を変更します。
  - **コピー:** 元の場所に残したまま、複製します。（`Ctrl + C`）
  - **切り取り（カット）:** 元の場所から移動させるために選びます。（`Ctrl + X`）
  - **貼り付け（ペースト）:** コピーまたは切り取りしたものを、別の場所に配置します。（`Ctrl + V`）
  - **削除:** ファイルやフォルダを「ごみ箱」に移動させます。（`Delete`キー）

# 文字入力操作

日本語を入力するには「IME (Input Method Editor)」というソフトウェアを使います。（Windows標準は Microsoft IME）

- **IMEのオン/オフ:**
  - キーボードの**「半角/全角 漢字」**キー（通常、左上）を押して、「あ」（日本語入力オン）と「A」（直接入力オフ）を切り替えます。
- **入力モードの切り替え:**
  - 日本語入力オン（「あ」の状態）で使います。
  - **ひらがな:** （初期状態）
  - **カタカナ:** `F7` キー（入力中）、または `無変換` キー
  - **英数字（全角/半角）:** `F9`, `F10` キー（入力中）、または `変換` キー
- **漢字変換:**
  - ひらがなを入力した後、**「Space（スペース）」**キーを押します。
  - もう一度押すと、別の候補が表示されます。
- **文節区切り直し:**
  - 意図しない場所で変換された場合（例：「きょうはいいてんき」→「今日歯医者天気」）
  - **「Shift」キーを押しながら「←」「→」キー**を押すと、文節の区切りを調整できます。

# ファイル保存

作成したデータは「保存」しないと消えてしまいます。（`Ctrl + S`）

## 💾 保存場所の種類

- **ローカル保存:**
  - 今使っている**PC本体の中**（CドライブやDドライブ、デスクトップ、ドキュメントフォルダなど）に保存することです。
  - そのPCでしか開けませんが、高速に動作します。
- **ネットワーク保存（ファイルサーバー）:**
  - 会社や学校内の**共有サーバー**に保存することです。
  - 許可された人であれば、ネットワーク内のどのPCからでもアクセスできます。
- **クラウド保存:**
  - **インターネット上**のサービス（Google Drive, OneDrive, Dropboxなど）に保存することです。
  - インターネットさえあれば、会社、家、スマホなど、**どこからでも**アクセスできます。他人との共有も簡単です。

## 📁 保存の基本

- **保存場所の指定:**
  - 「名前を付けて保存」をするとき、どこに保存するか（例: デスクトップ、ドキュメントフォルダ、USBメモリなど）を必ず意識して選びます。
- **ファイル名の付け方:**
  - 「いつ」「誰が」「何の」ファイルかが分かるような、**具体的で分かりやすい名前**を付けましょう。
  - （悪い例: `資料.xlsx`）
  - （良い例: `20251113_IT基礎知識_ネットワーク資料_v1.pptx`）

# インターネット基本操作

## 🌐 Webブラウザの使い方

Webサイトを見るためのソフトを「Webブラウザ」と呼びます。（Google Chrome, Microsoft Edge, Safari など）

- **ブラウザの起動:**
  - タスクバーやデスクトップにあるアイコンをクリックして起動します。
- **URLの入力:**
  - 画面上部のアドレスバーに、見たいサイトの住所（URL）を入力します。（例: `https://www.google.com`）
- **ハイパーリンク:**
  - Webページ上の、クリックできる文字や画像のことです。
  - クリックすると、関連する別のページに移動（ジャンプ）します。
- **ブックマーク（お気に入り）:**
  - よく見るWebサイトを登録しておく機能です。
  - アドレスバーの横にある「☆」マークなどで登録できます。
  - 次回から、ブックマーク一覧を選ぶだけでそのサイトにアクセスできます。

# ビジネスコミュニケーションツール

## 📧 メールの基本

- **宛先 (To):**
  - メールを**送る相手**（メインの受信者）のアドレスを入れます。「～様」
- **Cc (Carbon Copy):**
  - **参考・情報共有**のために見てほしい相手のアドレスを入れます。
  - Toの人にも、Ccに入れた人のアドレスは見えています。（「関係者各位」のような使い方）
- **Bcc (Blind Carbon Copy):**
  - ここに入れた相手にもメールは送られますが、**ToやCcの人からは見えません。**
  - お互いに面識のない複数の人に一斉送信する時など、プライバシー保護のために使います。
- **件名:**
  - メールの内容が**一目で分かる**ように、簡潔かつ具体的に書きます。（例: 「【ご相談】11/20の会議日程について」）
  - （悪い例: 「お疲れ様です」「こんにちは」）
- **ファイル添付:**
  - メールにファイル（Word, Excel, PDFなど）を付けて送る機能です。
  - 容量が大きすぎるファイル（数MB以上）は、クラウドストレージ（Google Driveなど）の共有リンクを送るのがマナーです。
- **ビジネスメールのマナー:**
  - 宛名（会社名、部署名、氏名）を正しく書く。
  - 冒頭の挨拶（「お世話になっております。」）と、結びの挨拶（「よろしくお願いいたします。」）を入れる。
  - 自分の署名（会社名、氏名、連絡先）を最後に入れる。

## 💬 チャットツールの利用

(Slack, Microsoft Teams, Google Chat など)

- **基本的な使い方:**
  - メールよりも**素早く、気軽な**コミュニケーションに使われます。
  - 特定の話題ごと（「プロジェクトA」「雑談」など）に「チャンネル」や「ルーム」が分かれていることが多いです。
- **メンション:**
  - `@ユーザー名` のように指定すると、その相手に通知を送ることができます。
  - 大勢がいるチャンネル内で、特に誰に見てほしいかを明確にするために使います。
- **リアクション:**
  - メッセージに対して絵文字（👍、👀、ありがとうございます）で反応する機能です。
  - 「見ました」「了解しました」といった簡単な返事の代わりに使うことで、やり取りを簡潔にできます。
- **スレッド（返信）:**
  - 特定のメッセージに対して「返信」することで、話題が混ざらないようにします。
  - 関連するやり取りがまとまるため、後から見返しやすくなります。
