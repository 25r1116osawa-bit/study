# 認証インターフェース
- インターフェース名：
  - Authentication
- プロパティ
  - id
    - number型
    - 読み込み専用
  - pass
    - string型
  - name
    - string型
- メソッド
  - login()
    - コンソールに「[name]さんログイン成功」と表示
    - 引数
      - 無し
    - 戻り値
      - void
# マネージメントインターフェース
- インターフェース名
  - Management
- メソッド
  - deleteUser
    - コンソールに「ID:[id]削除」と表示
    - 引数
      - id:number
    - 戻り値
      - void

# メンバークラス
  - 認証インターフェー スを実装
  - クラス名
    - Member

# 管理者クラス
- メンバークラスを継承し、マネージメントインターフェースを実装
- クラス名
  - Admin

# 動作確認として
- idを書き換えてコンパイルエラーが出ればOK

# テストプログラム
// インスタンスの作成と実行
Member型のメンバー1さんを作成
Admin型の管理者1さんを作成
メンバー1さんのログイン
管理者1さんのログイン
管理人1さんの権限でid:123を削除
