□読み物、記事の部分を作る &lt; article &gt;タグ
・ニュースサイトやブログサイトの記事部分のように独立して完結するような内容を書くことが多い

```
<article>
    <h2>記事title</h2>
    <p>最新のスマートフォン情報！新機種が発表されましたね！</p>
</article>
```
   </section>
<br><br>

　 <section>
□テーマを持ったグループを作る&lt;section&gt;タグ
・articleと似ているが、その部分だけを見ても完結しないひとまとまりのテーマを記載する

```
<section>
    <h2>その他のおすすめ記事</h2>
    <ul>
        <li><a href="#">スマートウォッチって使える？使えない？</a></li>
    </ul>
```
   </section>
<br><br>


   <section>
□ページのメインコンテンツ部分を作る<main>タグ
そのページの核となるコンテンツ全体は<main>タグで囲みます、その中に様々なグループが入ることがあります。

```
<main>
    <article>
        <h2>記事title</h2>
        <p>最新のスマートフォン情報！新機種が発表されましたね</p>
    </article>
    <section>
        <h2>そのほかのおすすめ</h2>
            <ul>
                <li><a herf="a">スマートウォッチって使える？使えない？</a></li>
                <li><a herf="a">ランニングにはこのガジェットがおすすめ</a></li>  
    </section>
</main>

```
   </section>
</main>
