## 埋め込みタグ
- `<iframe>`タグ
  - 他のHTMLドキュメントを埋め込むためのタグ
  - 例：
    ```html
    <iframe src="https://www.example.com" width="600" height="400" frameborder="0"></iframe>
    ```
- 様々なコンテンツを埋め込むことができる
  - YouTube動画
  - Googleマップ
  - 他のウェブページ
- 利用規約を確認することが重要

## セレクター
- CSSで特定のHTML要素を選択するためのパターン
- 主なセレクターの種類
  - タグセレクター
    - 例：`p { color: blue; }`
  - クラスセレクター
    - 例：`.classname { font-size: 16px; }`
  - IDセレクター
    - 例：`#idname { margin: 10px; }`
  - 属性セレクター
    - 例：`a[href^="https"] { text-decoration: none; }`
  - 擬似クラスセレクター
    - 例：`a:hover { color: red; }`
  - 擬似要素セレクター
    - 例：`p::first-letter { font-size: 200%; }`
  - 子セレクター
    - 例：`div > p { margin: 0; }`
  - 子孫セレクター
    - 例：`div p { line-height: 1.5; }`
  - グループセレクター
    - 例：`h1, h2, h3 { font-family: Arial; }`
  - ユニバーサルセレクター
    - 例：`* { box-sizing: border-box; }`
