```
CSS
/* float を使った横並び */
#float ul {
    overflow: hidden; /* 回り込み解除 */
}

#float li {
    float: left;              /* 横並び */
    list-style: none;         /* 行頭記号を消す */
    border: 1px solid #777;   /* 罫線 */
    width: 20%;               /* 5つで100% */
    box-sizing: border-box;   /* 枠線と padding を幅に含める */
}
```

<h2>2.inline-block を使った横並び</h2>

<p>要素の分類変更で作成（inline-block）
li をインラインブロック要素に変更し、横に並べる方法。</p>
    
```
CSS
/* inline-block を使った横並び */
#inline ul {
    font-size: 0; /* 要素間の半角スペースを消す */
}

#inline li {
    display: inline-block;       /* 横並び */
    border: 1px solid #777;
    width: calc(100% / 7);       /* 7等分 */
    text-align: center;
    font-size: 16px;             /* 復元 */
    box-sizing: border-box;
}

#inline a {
    display: block;              /* リンク領域を広げる */
    background-color: #777;
    color: #fff;
    text-decoration: none;
}
```

<h2>3. Flexbox を使った横並び（推奨）</h2>

```
CSS
/* フレックスボックス */
#flex ul {
    display: flex;

    /* アイテム間の余白（justify-content） */
    justify-content: space-evenly;
    /* gap: 縦方向 横方向 */
    gap: 1em 2.5%;
}

#flex li {
    list-style: none;
    border: 1px solid #777;
    width: 20%;              /* 5つで100% */
    text-align: center;
}

```
