## HTMLフォーム
- ウェブサイトでユーザーから情報を集め、それをサーバー（サイトの裏側）に送るための機能です。
- 例：
  - 会員登録、ログイン、お問い合わせ、注文。

### フォームの基本的な構成要素
- formタグ
  - フォームの枠組み
  - どこにデータを送信するのか？
    - action属性で指定(データの送信先のURL)
  - どうやってデータを送信するのか？
    - method属性で指定(データの送信方法)
      - GET:検索用で取得した情報をURLに埋め込まれる
      - POST：電話番号や住所など個人情報などを表示したくない場合に利用
- inputタグ
  - 入力部品
    - サーバーに送信する情報
    - type属性で部品の種類を変更できます。
    - 亜種として
      - textareaタグ
      - selectタグ / optionタグ
- name属性
  - サーバーにデータを送信するとき、そのデータが何のデータなのか識別するための名前
  - すべての入力部品に必須です。

### 主要な入力部品
- type属性を変えることで、さまざまな入力形式を実現します。

| `type`の値 |  表示される部品 | 説明 |
| :--- | :--- | :--- |
| **`text`** | 1行のテキスト入力欄 | 名前や住所など、一般的な文字を入力してもらう場所。 |
| **`password`**| パスワード入力欄 | 入力した文字が「\*」や「•」で隠される入力欄。 |
| `checkbox`| チェックボックス | 複数ある選択肢の中から、いくつでも選べる部品。 |
| `radio` | ラジオボタン | 複数ある選択肢の中から、どれか一つだけを選べる部品。 |
| `submit`| 送信ボタン | フォームの入力を**サーバーへ送る\*\*ためのボタン。 |
