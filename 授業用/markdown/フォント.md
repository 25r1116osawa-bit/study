font-family とは？

🔤 font-family ＝ 「文字の見た目（フォント）を指定するCSSプロパティ」

Webサイトで表示される文字は、フォントによって見た目が変わります。
そのフォントを どれで表示するか指定するのが font-family です。

1. 基本の書き方
セレクタ {
  font-family: フォント名;
}


例：

.font_test {
  font-family: serif;
}

2. 複数フォントの指定（フォールバック）

「使えなかったら次のフォントを使う」という指定ができます。

font-family: "Arial Black", serif;


最初のフォントが優先

日本語対応していないフォントもあるため、複数指定が基本

3. まず英語フォントを先に書こう

日本語フォントは英数字にも対応していることが多いです。
先に日本語フォントを書いてしまうと、英語フォントが使われません。

❌ NG

font-family: serif, "Arial Black";


⭕ OK

font-family: "Arial Black", serif;

4. 代表的なフォントファミリー（総称）
フォント名	表示の特徴	日本語対応
serif	明朝体	あり
sans-serif	ゴシック体	あり
cursive	筆記体	なし
fantasy	装飾系	なし
monospace	等幅フォント	なし

迷ったら serif または sans-serif が無難！

5. OS の違いに注意

Windows と Mac では 使えるフォントが違う

指定したフォントが入っていないと適用されない

➡ 同じ見た目にしたいなら Web フォントを使うのがおすすめ！

6. Webフォントとは？

Web上にあるフォントを読み込む仕組み。
OSに依存せず、同じフォントで表示できる。

代表例：

Google Fonts（無料）

モリサワ（有料・高品質）

7. Webフォントの使い方（Google Fonts 例）

① フォントを読み込む（HTML または CSS）

HTML の場合

<link href="https://fonts.googleapis.com/css?family=BioRhyme+Expanded" rel="stylesheet">


CSS の場合

@import url('https://fonts.googleapis.com/css?family=BioRhyme+Expanded');


② font-family で指定

.font_test {
  font-family: 'BioRhyme Expanded', serif;
}
