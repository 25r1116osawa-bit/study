---加湿器の仕様書
### ■ クラス
- クラス名
  - Humidifier（加湿器）

### クラスの説明

空気中に水分を放出する家電。
電源ON/OFF、加湿量の調整、水切れ検知などの機能を持つ。

### ■ プロパティ
- power
  -   電源状態（ON / OFF）を保持する。

tankCapacity

タンクの最大水容量（例：3.0L）。

currentWater

現在タンクに入っている水量。

mode

加湿モード（弱 / 中 / 強 / オート）。

humidity

現在の湿度（センサーがあるモデルの場合）。

isWaterEmpty

水切れ状態を示すフラグ（true / false）。

■ メソッド
turnOn()

加湿器の電源をONにする。

turnOff()

加湿器の電源をOFFにする。

startHumidifying()

加湿を開始する（ミストを出す）。

stopHumidifying()

加湿を停止する。

setMode(mode)

加湿モード（弱・中・強・オート）を設定する。

refillWater(amount)

指定量の水をタンクに補充する。

detectWaterLevel()

現在の水量を確認し、水切れなら isWaterEmpty を true にする。

getStatus()

現在の状態（電源、水量、湿度、モードなど）をまとめて返す。
