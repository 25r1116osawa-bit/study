# 概要

**HTML**
→テキストにタグと呼ばれる印をつけて、それぞれの部分が何であるかを明示する。

**CSS**
→HTMLの印によって示された各範囲の表示方法を指定する。

イメージ画像①：
![alt text](image-1.png)

ブロックという要素で管理し、ブロックの中にブロックを入れることや、ブロック同士を横に並べたりすることができる。

htmlのフォーマットは下記



<!DOCTYPE html>
<html lang="ja">
<head>
<meta charset="UTF-8">
<title></title>
</head>
<body>
</body>
</html>
